# Last CST1510 module project
To run the program, write "cd Main" on a dedicated terminal and then use the command "streamlit run main.py". 
Create your own API key on groqCloud, copy it and paste in the three files: Cyber_Incidents_Dashboard.py, Metadata_Dashboard.py and It_Ticket Dashboard.py, on line 8 in the variable "client".
The secrets.toml does not need to be created, it will be generated once the program is run. In it, you can paste your API key.
Python version should be 3.9 and above. 
To install the required libraries, use the command term "pip". The list of the required libraries is found in "Requirements.txt"
This repo is public. To access the URL, click the "<>Code" button near "Add file" and copy the URL. In VScode, in the searchbar, write >clone and paste the URL to access the project.
